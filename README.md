# 学生成绩管理系统

#### 介绍
学生成绩管理系统基于B/S（Browser/Server）模式开发，系统设计采用了OOP思想，系统开发选择了Eclipse 2020-06作为集成开发环境，使用了JSP和Java语言编写程序；前端使用HTML+CSS+JavaScript技术设计开发页面；后台数据库使用MySQL5.6.26。
